## About

### Course Overview

Introduction to Software Systems (ISS) is a foundation course designed to introduce concepts associated with developing software systems to early undergraduate students. Basic concepts about Operating Systems, Computer Networks, Software Engineering principles, and Databases are covered as part of this course. 

### Topics Covered

Programming languages and tools like BASH SHELL, SQL, NoSQL, HTML, CSS, JavaScript, JS Frameworks, APIs, REST, Python, and Flask are covered as part of this course.
